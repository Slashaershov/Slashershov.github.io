# Slashershov.github.io

Привет чтец, я - разработчик игр на Unity. Здесь ты можешь прочитать список проектов в которых мне удосужилось поучаствовать, и которые мне не стыдно показать.

** Флагманом является Rusty Rangers (https://store.steampowered.com/app/1599420/Rusty_Rangers/). Ссылку на гит не предоставлю, ибо это коммерческий проект со своим НДА. Игра представляет собой рогалик-Метроидвания, с ОГРОМНОЙ кодовой базой на 5 лет разработки, в нем я побывал почти в каждой секции по той или иной задаче. Краткий список действий в этом проекте таков:

1) добавление нового функционала:
- новые игровые режимы (BossRush, TrainMode, Arena)
- создание новых врагов в игру (логика поведения, настройка аниматоров, прокидывание конфигов)
- создание боссов с нуля (6 новых боссов, в том числе ласт босс игры)
- контент паки с предметами
- создание расширений редактора для нужд проекта

2) Доработка существующего функционала:
- рефакторинг legacy систем
- проработка взаимодействия комбинаций предметов в рогалике
- полировки логики противников и боссов
- верстка UI и in-gameUI
- улучшение системы навигации для ИИ
- расширение формата сохранения игры в protobuf

3) Оптимизация игры:
- внедрение системы пулинга (снарядов, противников, предметов, vfx-ов)
- вычищение утечек памяти
- профилирование производительности и сглаживание спайков
